# PhoenGo — Engagement system: design + build plan

> Рабочий документ. Составлен 2026-06-26 из мульти-агентного дизайн-разбора (17 линз:
> Duolingo-удержание, Apple-craft/моушн, гейм-прогрессия/экономика, SDT-мотивация,
> нарратив, тин-айдентика, глаза 11-летки, визионер, доступность, скептик).
> Цель: довести феникс/XP/уровни/стрики до уровня Duolingo×Apple, с фишкой, которой
> нет ни у кого, чтобы тинейджеры залипали и хвастались. Один файл, без тяжёлых ассетов,
> reduced-motion безопасно, **никогда не ломать сохранённый прогресс**, growth-mindset.

## Видение (одной фразой ребёнка)
> «Мой феникс выкован из всех слов, что я выучил; я подкидываю ему немного огня каждый
> день, чтобы он поднялся; а когда я отстаю — он не умирает, он отдыхает в пепле моих же
> слов, пока я снова его не зажгу.»

Статус (XP/ранг/коллекция) **только растёт** → ощущается безопасно. Сегодняшний огонь
(дневная цель/стрик) — добрый и защищён заработанным щитом. Компетентность (embers)
зарабатывается сложностью → число значит навык, а не время. Ничего не стыдит, ничего не
продаётся, ничего не ломает сейв.

## Сигнатурная фишка (аналога нет): **ASHES REMEMBER**
Перерождение, собранное из **собственных выученных слов ребёнка**. Когда феникс в пепле —
именно слова, которые ты освоил, и есть тлеющее ложе из углей, не потеря. Каждый верный
ответ поднимает уголёк и заново зажигает крыло (одна CSS-переменная `--ash-mix` 0→1 гонит
фильтр существующего phoenix.png от пепла к огню — тот же арт, без новой графики). На 10-м
верном: затемнение → 400 мс паузы → радиальная огненная маска → фильтр в полную жизнь →
кольцо углей → текст «Nothing you learned was lost. You rose from it.» + вечная метка
«Risen ×N». Reduced-motion: мгновенная вспышка цвета + статичный список 10 зажжённых слов +
«Risen.» Непрерабатываемо: нужны ОБА актива — метафора феникса-из-пепла И хранилище реальных
выученных слов ребёнка. ~90% уже в коде; не хватало только провода (рекордер 10 слов +
смещение выбора к самым старым/сильным освоенным словам).

## Принципы (железные)
1. **Сначала провод, потом украшение люка:** сигнатурная фича была написана и недостижима —
   сначала чиним ловушку Ash, потом новый «вкус». ✅ сделано.
2. **Атомарные коммиты:** разметка + проводка в одном коммите — живой home никогда не
   ломается между пушами. `/board` на home перед каждым визуальным пушем.
3. **Save-safe:** каждое новое поле инициализируется как `if (d.X===undefined) d.X=default`;
   старый сейв обязан грузиться и рисовать home корректно.
4. **Один писатель на число:** `addXP` — единственный писатель XP, `LEVELS` — единственная
   кривая; у embers свой единственный писатель + реальный сток. XP никогда не тратится.
5. **Без стыда, без тёмных паттернов, ничего не продаём:** только forward-looking копи; ни
   таймеров «теряешь стрик», ни вины. Каждую строку — через `/kid-content`.
6. **Reduced-motion и фокус по построению:** гейтим моушн в ОДНОЙ точке (spawnConfetti /
   celebrate), не по сайтам; каждый интерактив — все состояния.
7. **Один файл, без нового веса:** только инлайн CSS/SVG/JS; анимируем существующий
   phoenix.png через CSS-переменные, не новые PNG.
8. **Заслужи место против «учит ли ребёнок слово и возвращается ли»:** если идея не оживляет
   феникса, не делает камбэк достижимым и не закрывает честную дневную петлю — ждёт.

- **#7 Feed the Flame — дневная цель-кольцо (хребет удержания)** [M] — второе кольцо на

- **#9 Ember Shield — добрая защита стрика** [M] — «анти-Duolingo»: щит зарабатывается
  УЧЁБОЙ (не покупкой), молча спасает стрик при пропуске, без таймеров/вины. Зависит от #1/#7.
- **#10 Один айдентити** [S] — схлопнуть два конфликтующих набора имён (LEVELS.title

  замедлились.
- **#12 Палитра феникса на профиль + Flame Card** [M] — сейчас 3 профиля делят одинаковые
  конфетти/градиент (карточка ребёнка не отличается от брата). Дать каждому свою ауру + по

  Это и есть «показать друзьям». Идентичность-флекс, без лидербордов.
- **#13 Prestige Aviary + ровный ритм эволюции** [L] — на капе Blazing Phoenix уходит в
  постоянную коллекцию, начинается новое яйцо (ранг/XP не сбрасываются). Чинит инвертированные
  гейты эволюции (стадия 4 требует 7-дн стрик, стадия 5 — 0). Самый большой; последним.

### ✂️ Вырезано (сознательно)
- Любые таймеры «теряешь стрик»/срочность/«не потеряй щит» — тёмный паттерн, тинейджеры чуют.
- Продажа чего-либо (платный freeze, косметика, платные открытия) — earned-not-bought = вся
  достоверность для этой аудитории.
- Лидерборд/соц-ранжирование/сравнение с друзьями — тревожит 9–14. Флекс-карточка (что
  отправляешь сам) — да; лестница — нет.
- 5 PNG-артов стадий / по-мировые существа — phoenix.png 2.5МБ; пять = 12МБ+. Формы — CSS/SVG.
- Второй писатель XP / трата против XP-баланса — XP только растёт; тратимое — отдельные embers.
- Хаптика/звук — iPad игнорит vibrate; только как opt-in stretch, не вытесняя P0/P1.
- Запуск дневного кольца + щита + хаптики + карточки + косметики разом пятью полуфичами —
  «широта как прокрастинация», которая оставляет home сломанным между коммитами. Кардинальный грех.
